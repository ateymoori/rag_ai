This documentation will try to explain how we try to sort objects inside Netbox. It should also help you when you need to add an asset in Netbox. You login with your normal AD account, to get login access you need to be in the AD group app-netbox-users . This will only give you login access, to be able to see all the objects you need to be in the app-netbox-reader AD group. If you need to contribute and add devices and such you, you need to be in the app-netbox-contributor AD group.
